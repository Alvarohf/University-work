# Java projects in spanish
Here are all the projects created in spanish at UAH in Computer Engineering degree.
* **Poker** poker game with a custom GUI with collections an serial data.
* **ConsumidorProductor** producer/consumer program wich uses concurrency programming and has a GUI.
* **RMIServidor** simulates museum wich has to be accessed by different clients using RMI and concurrence programming.
* **SocketsMultiThreadServidor**  simulates a gas station wich has multiple clients that connect through TCP sockets and using concurrence programming. It also has a custom GUI for clients and server. Here clients could watch what is happening in the server.

# Proyectos de java en español
Aquí están todos los proyectos desarrollados en la UAH en el grado de Ingeniería Informática.
* **Poker** juego de poker con una GUI propia que emplea colecciones y serializa los datos.
* **ConsumidorProductor** programa de productor/consumidor que emplea programación concurrente y tiene una GUI.
* **RMIServidor**  simula un museo al que acceden diferentes clientes empleando RMI y programación concurrente.
* **SocketsMultiThreadServidor**  simula una gasolinera que tiene múltiples clientes que se conectan empleando sockets TCP y se usa además programación concurrente. Tiene también una GUI propia para el cliente y servidor. Aquí el cliente podrá observar lo que sucede en el servidor.
